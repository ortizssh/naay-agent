AI-powered shopping assistant for WooCommerce. Intelligent product search, recommendations, and cart management through a chat widget.

== Description ==

Kova Agent is an AI-powered shopping assistant that helps your customers find products, get personalized recommendations, and manage their shopping cart through natural conversation.

= Features =

* **Intelligent Product Search** - Customers can search for products using natural language
* **Personalized Recommendations** - AI-powered product recommendations based on browsing behavior
* **Cart Management** - Add, remove, and update cart items through chat
* **Semantic Search** - Understands product descriptions and finds relevant matches
* **Multi-language Support** - Works with customers in their preferred language
* **Customizable Widget** - Match the chat widget to your store's branding
* **Analytics** - Track chat conversions and AI-assisted sales

= Requirements =

* WordPress 5.8 or higher
* WooCommerce 5.0 or higher
* PHP 7.4 or higher
* Kova Agent account (sign up at https://kova.ai)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/kova-agent`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Kova Agent' in the admin menu
4. Enter your WooCommerce API credentials
5. Click 'Test Connection' to verify
6. Click 'Sync Products' to import your catalog
7. Enable the widget and save settings

= Generating WooCommerce API Keys =

1. Go to WooCommerce > Settings > Advanced > REST API
2. Click 'Add key'
3. Set Description: "Kova Agent"
4. Set User: Select your admin user
5. Set Permissions: "Read/Write"
6. Click 'Generate API key'
7. Copy the Consumer Key and Consumer Secret to the Kova Agent settings

== Frequently Asked Questions ==

= Do I need a Kova Agent account? =

Yes, you need a Kova Agent account to use this plugin. The plugin connects to Kova's AI backend to provide intelligent shopping assistance.

= Is my data secure? =

Yes, all data is transmitted securely via HTTPS. Product data and customer conversations are stored securely and never shared with third parties.

= Can I customize the chat widget? =

Yes, you can customize the widget position, color, welcome message, and title to match your store's branding.

= Does it work with variable products? =

Yes, Kova Agent fully supports WooCommerce variable products and can help customers find the right product variation.
