🤖 Naay Widget Demo

Interactive demo of the Shopify AI chat widget

Widget Configuration

How to Test

  1. Fill in your Shopify store domain and backend API endpoint
  2. Customize the appearance and behavior settings
  3. Click "Initialize Widget" to create the chat widget
  4. The widget will appear as a floating button in the specified position
  5. Click the button to open the chat and start testing

Features to Test

  • Product search with natural language queries
  • Product recommendations based on context
  • Cart integration (if cart ID is available)
  • Responsive design (try on mobile)
  • Dark/light theme switching
  • Accessibility features (keyboard navigation)
  • Error handling (try with invalid API endpoint)
  • Session persistence across page reloads

Sample Queries to Try

  • "Show me red dresses under $100"
  • "I need running shoes for women"
  • "What's on sale this week?"
  • "Add a blue t-shirt to my cart"
  • "What are your return policies?"
  • "Recommend something similar to my last purchase"
